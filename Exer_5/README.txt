Exer_5 - Huong dan chay test

1. Cau truc
- `Tests.slnx`: solution cua bai Exer_5
- `Exer_1_2.Tests`: test cho Exer_1 va Exer_2
- `Exer_3.Tests`: test cho Exer_3

2. Yeu cau
- Visual Studio 2022
- .NET SDK phu hop voi project
- Co san source cua `Exer_1`, `Exer_2`, `Exer_3` cung cap voi `Exer_5`

3. Cach mo
- Mo folder `Exer_5`
- Mo file `Tests.slnx`

4. Cach chay
- Restore NuGet packages neu Visual Studio yeu cau
- Build solution
- Mo `Test Explorer`
- Chon `Run All`

5. Noi dung test
- `Exer_1_2.Tests`
  Test source-based cho Exer_1 va Exer_2 de tranh phu thuoc DB, Session va web runtime
- `Exer_3.Tests`
  Test login page model cua Exer_3
